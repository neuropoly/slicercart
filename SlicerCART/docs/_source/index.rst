About SlicerCART
-----------

SlicerCART is a module in 3D Slicer software aimed to facilitate large
datasets image visualization, manual segmentation and classification tasks.
SlicerCART is based on a dynamic graphical user interface (GUI) so the user
can edit the segmentation or classification labels for a project-specific
configuration design. The scope extends to 3D images quality-control
workflows in imaging studies.

More information can be found :doc:`here <purpose>`.

For a complete list of functionalities, see :doc:`functionalities`.
