Getting Started
-------------

Before using SlicerCART, make sure you have:

1. Installed 3D Slicer
2. Installed the SlicerCART module
3. Prepared your image data
4. Created a project configuration file

Basic Workflow
------------

1. **Project Setup**
   
   * Create or load a project configuration
   * Configure segmentation labels
   * Set up classification categories

2. **Data Loading**
   
   * Select your volume folder
   * Choose output directory
   * Load existing segmentations (if any)

3. **Segmentation**
   
   * Use painting tools
   * Apply interpolation
   * Save segmentations

4. **Classification**
   
   * Add classifications
   * Review and modify
   * Export results

5. **Quality Control**
   
   * Compare segmentation versions
   * Review classifications
   * Export statistics
